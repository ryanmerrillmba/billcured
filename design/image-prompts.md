# Image Prompts for NanoBanna

This file contains prompts for generating images needed for the BillCured lead generation pages.

---

## Image 1: 3D Ebook Mockup

**File:** `public/images/billing-blind-spot-mockup.png`
**Specs:** 800x600px, transparent background, slight shadow

### Prompt

```
Create a professional 3D ebook/PDF mockup for a guide called "The Billing Blind Spot".

Specifications:
- 800x600 pixels, transparent background
- Slight realistic shadow beneath
- Book angled 15-20 degrees for depth
- Clean, modern cover design

Cover content:
- Title: "The Billing Blind Spot" in bold, professional font
- Subtitle: "What Hospitals Don't Know About Your Bill"
- Small "BillCured" logo at bottom
- Color scheme: Teal (#2B6B7C) primary, coral (#FA8072) accent
- Clean, minimal design - no busy graphics
- Maybe subtle dollar sign or medical cross watermark

Style: Professional, trustworthy, empowering. Similar to business/finance ebook mockups.
```

## Image 2: Hero Background Image

**File:** `public/images/free-guide-hero.jpg`
**Specs:** 1920x1080px, slightly desaturated for text overlay

### Prompt

```
Create a hero background image for a medical bill negotiation website.

Scene: A middle-aged person (30s-50s, diverse representation welcome) on a phone call, looking confident and empowered. They're in their home - kitchen or home office setting. Natural, warm lighting.

Mood: Hopeful, empowered, relieved - like they just won a negotiation. NOT stressed or worried.

Specifications:
- 1920x1080 pixels
- Slightly desaturated/muted to work as text background
- Leave clear space on the LEFT SIDE for text overlay
- Subject positioned on right third of image
- Warm color tones (not cold/clinical)
- Soft depth of field on background

Style: Authentic, relatable, editorial photography style. Not overly stock-photo perfect.
```

## Image 3-5: Benefit Icons (Set of 3)

**Files:**
- `public/images/icon-hidden-truth.svg`
- `public/images/icon-missing-piece.svg`
- `public/images/icon-one-question.svg`

**Specs:** 120x120px, SVG preferred, consistent style

### Prompt

```
Create a set of 3 matching icons for a medical billing website.

Style specifications:
- 120x120 pixels each
- Line icon or flat design style
- Primary color: Teal (#2B6B7C)
- Accent color: Coral (#FA8072) for highlights
- Consistent stroke weight and style across all 3
- Clean, modern, professional

Icon 1 - "Hidden Truth":
Concept: Revealing hidden errors in documents
Visual: Magnifying glass over a document with error marks/highlights visible

Icon 2 - "Missing Piece":
Concept: Information gap that billing departments don't have
Visual: Document with a puzzle piece missing OR document with a gap/void in it

Icon 3 - "The One Question":
Concept: A simple question that changes everything
Visual: Speech bubble with question mark OR phone handset with a lightbulb

Make them feel cohesive as a set - same visual weight and complexity.
```

---

## Usage Notes

1. Generate each image using the prompts above
2. Save to the specified file paths in `/public/images/`
3. The mockup image needs a transparent background (PNG)
4. The hero image can be JPG for smaller file size
5. Icons should be SVG for scalability, or PNG if SVG not available

## Color Reference

- **Primary (Teal):** #2B6B7C
- **Accent (Coral):** #FA8072
- **Background:** #F8FAFC
- **Text:** #1A1A2E
